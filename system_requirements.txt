Для работы программы требуется следующий состав программных средств и характеристик:

+ Операционная система Ubuntu 22.04 или MacOS Sonoma;
+ 4 ядра CPU, 16 ГБ RAM, 50 ГБ свободного дискового пространства. Минимальные требования необходимы для обеспечения корректной работы PostgreSQL, RabbitMQ, Redis, MinIO и всех микросервисов (auth, backend, render, chat, project, storage), с одновременной обработкой нескольких десятков запросов, а также для выполнения ML-операций генерации 3D-контента с использованием PyTorch и модели ObLLoMov;
+ Видеокарта с поддержкой CUDA (рекомендуется 8+ ГБ VRAM) для ускорения ML-генерации, либо CPU с AVX2 для работы в режиме CPU-inference;
+ Docker версии 24.0+ и Docker Compose версии 2.20+ для контейнеризации и оркестрации микросервисов;
+ Любой современный браузер (Chrome 90+, Firefox 88+, Safari 14+) с поддержкой WebGL 2.0 для корректной работы Three.js и 3D-визуализации;
+ Стабильное интернет-соединение (минимум 25 Мбит/с) для быстрой загрузки 3D-моделей из объектного хранилища, передачи данных между клиентом и микросервисами, а также для streaming-генерации через WebSocket.
